# LUT-ALMs: Trading Off Accuracy and Power for Approximate Logarithmic Multipliers via LUT Optimization
This repository contains the official code release for LUT-ALM and MM-ALM, the logarithmic multipliers are presented in the paper "LUT-ALM: Trading Off Accuracy and Power for Approximate Logarithmic Multipliers via LUT Optimization". This repository is used for open-source paper-related code.
# Repository Structure
1. ./lut-generate/: the generation code of LUTs in LUT-ALM and MM-ALM in MATLAB code
2. ./software_code/: the software code of LUT-ALM and MM-ALM
3. ./README.txt: the general readme of this repository
